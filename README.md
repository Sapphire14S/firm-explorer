# Firm Explorer

Firm Explorer is a React-based web application for searching companies and exploring key information through a clean and interactive interface.

## Live Demo

https://firm-explorer.vercel.app/

## Features

* Search companies with real-time suggestions
* View company summaries and basic details
* Access official company links
* Responsive and modern UI
* Light and Dark mode support

## Tech Stack

* React
* Vite
* React Router
* CSS
* Wikipedia API
* Clearbit Logo API

## Installation

```bash
git clone https://github.com/Sapphire14S/firm-explorer.git
cd firm-explorer
npm install
npm run dev
```
